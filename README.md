# 🏡 Villa & Jardin — Site de location saisonnière

Site web élégant pour la location de votre villa 130 m² avec piscine et jardin paysager.

## 📁 Structure du projet

```
/
├── index.html          ← Site principal (tout en un fichier)
├── photos/             ← 📷 Mettez vos photos ici !
│   ├── facade.jpg
│   ├── piscine.jpg
```

## 📷 Ajouter vos photos

1. Créez un dossier `photos/` à la racine du projet
2. Copiez vos photos dedans avec les noms ci-dessus
3. *(optionnel)* Pour renommer : ouvrez `index.html` et cherchez `data-src="photos/..."` pour changer les noms

**Formats acceptés :** JPG, PNG, WebP — Taille recommandée : 1200×800 px minimum

## 🚀 Publier sur GitHub Pages (gratuit)

1. Créez un compte sur [github.com](https://github.com) si ce n'est pas déjà fait
2. Créez un nouveau repository (ex : `ma-villa`)
3. Uploadez tous les fichiers (index.html + dossier photos)
4. Allez dans **Settings → Pages**
5. Choisissez la branche `main` et dossier `/ (root)`
6. Cliquez **Save** — votre site sera en ligne sur `https://votre-pseudo.github.io/ma-villa`
